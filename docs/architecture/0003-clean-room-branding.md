# ADR 0003: Clean-room product and branding

Status: accepted

## Decision

The product name is **Open Data Fusion**. Its source, API contract, UI, wording, marks, and test corpus are independently created. Public product documentation may describe comparable industrial outcomes, but must not imply affiliation, endorsement, API compatibility, or shared implementation with Cognite.

Dependencies require version-specific license, NOTICE, vulnerability, and distribution review before release.
